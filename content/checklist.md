## 受診前の症状メモ

[受診]({{< relref "/iryo/jushin-nagare.md" >}})の限られた時間で医師に正確に伝えるためのメモです。**本人の見栄やその日の調子に左右されない「普段の記録」**が、診断のいちばんの手がかりになります。

- [ ] 気になる症状（もの忘れ・言動の変化など）を具体的に3つ
- [ ] それぞれ「いつごろから」「どのくらいの頻度で」
- [ ] いちばん最近の具体的なエピソード（日付と内容）
- [ ] 以前と比べて「できなくなったこと」（料理の段取り、運転、金銭管理など）
- [ ] 現在の病気と飲んでいる薬（**お薬手帳を持参**）
- [ ] 家族に認知症の人がいるか
- [ ] 生活の様子（睡眠・食欲・外出頻度・飲酒）
- [ ] 本人が「困っている」と言っていること

## 介護保険申請の持ち物

[申請の流れ]({{< relref "/seido/kaigo-hoken.md" >}})の最初のステップです。窓口は市区町村の介護保険課または地域包括支援センター。

- [ ] 介護保険被保険者証（65歳になったとき郵送で届いた保険証）
- [ ] 健康保険証（40〜64歳の方の場合）
- [ ] マイナンバーがわかるもの
- [ ] 本人確認書類（申請に行く人のもの）
- [ ] かかりつけ医の名前・医療機関名・連絡先のメモ（主治医意見書の依頼に必要）
- [ ] 印鑑（自治体による）
- [ ] 普段の困りごとを書いたメモ（認定調査で調査員に渡すため。上の「症状メモ」が使えます）

## 施設見学チェック

[施設選び]({{< relref "/seido/shisetsu-erabi.md" >}})の見学時に。できれば食事の時間帯に、2〜3施設を見比べてください。

**空気を見る**

- [ ] 入居者の表情は穏やかか。職員の言葉づかいは丁寧か（子ども扱いしていないか）
- [ ] 廊下・トイレのにおい
- [ ] 入居者が部屋に閉じこもっていないか。共用スペースに人の気配があるか

**体制を聞く**

- [ ] 夜間の職員体制（人数）
- [ ] 認知症の症状（大声・ひとり歩きなど）が強く出たときの対応方針
- [ ] 医療対応の範囲（協力医療機関、看取りまで対応するか、入院になったら退去か）
- [ ] 職員の定着率・研修（答えの具体性を見る）

**お金を確認する**

- [ ] 月額費用に「含まれないもの」（おむつ代・医療費・理美容・レク費）
- [ ] 入居一時金の返還ルール
- [ ] 要介護度が上がったときの費用の変化と退去要件

## もしもの備え（ひとり歩き・災害）

[ひとり歩きへの備え]({{< relref "/care/hitoriaruki-sonae.md" >}})とあわせてどうぞ。

**身元と連絡**

- [ ] 衣類・靴・持ち物に名前と連絡先を付けた
- [ ] 財布・定期入れに連絡先カードを入れた
- [ ] 最近の写真（顔・全身）をスマホに保存した
- [ ] 市区町村の見守りSOSネットワークに登録した
- [ ] ご近所・よく行く店に事情を伝えた

**災害への備え**

- [ ] 薬とお薬手帳のコピーを非常持ち出し袋に（**薬の情報は命綱です**）
- [ ] 避難先で本人が落ち着ける物（写真・なじみの小物）を用意した
- [ ] 自治体の避難行動要支援者名簿への登録を確認した（窓口: 市区町村）
- [ ] 紙パンツ・とろみ剤など本人専用の消耗品を数日分

## 入院時に病棟へ渡すメモ

[入院への備え]({{< relref "/iryo/nyuin-sonae.md" >}})より。入院時に看護師に渡すと、ケアが大きく変わります。

- [ ] 認知症であること・普段の生活の自立度
- [ ] 呼ばれ慣れている呼び名・安心する話題・不安なときのサイン
- [ ] 補聴器・眼鏡・義歯の有無
- [ ] 飲んでいる薬すべて（お薬手帳）
- [ ] 家族の連絡先（日中つながる番号）

*チェック項目は一般的な目安です。個別の状況については、地域包括支援センターやケアマネジャーにご相談ください。*
